Welcome to NeXDaTaS's documentation!
====================================

This project contains the following repositories:

-  `Nexus Data Writer <home.html#nexus-data-writer>`_ implemented as a Tango
   server
-  `Configuration Tool <home.html#nxs-component-designer>`__ written in PyQt ,
   NXS Component Designer
-  `Configuration Server <home.html#nxs-configuration-server>`__ implemented as
   a Tango server on a MySQL database
-  `Nexus Command-line tools <home.html#nexus-tools>`__ provides fast access to
   Configuration Server and Data Writer from command-line interface
-  Simple examples of configuration files and control clients

`To install Debian packages ... <home.html#install-debian-packages>`__
