Welcome to NeXDaTaS’s documentation!¶

This project contains the following NeXus software:

  • Data Writer - writer implemented as a Tango server

  • Configuration Server - MySQL database frond-end implemented as a Tango server

  • Selector Server - configuration configuration selector implemented as a Tango server

  • Component Designer - GUI configuration tool written in PyQt

  • Component Selector - GUI component selector written in Taurus (PyQt)

  • NeXus Macro GUI - Macro GUI with the additional Component Selector widget written in Taurus (PyQt)

  • Sardana Recorder - NeXus Sardana Recorder

  • Command-line tools - scripts providing fast access to Configuration Server and Data Writer from command-line interface

  • Simple examples of configuration files and control clients
